# Semaforo
Programa de prueba de funcionamiento de un semaforo  

## Modo Normal
17 Segundos en verde  
3 Segundos parpadeando verde  
3 segundos amarillo  
2 segundos rojo  

## Modo intermitente
Parpadea cada segundo el amarillo y el display
